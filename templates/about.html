About This Project

This web application is a demonstration of Steem Store built for the Systems Development unit. It showcases the integration of relational data, NoSQL storage, secure authentication, and cloud-based services using modern web development practices.

The system is powered by:

  • Flask for the web framework
  • Google Cloud SQL (MySQL) for structured game and user data
  • MongoDB Atlas for unstructured user reviews
  • Google Cloud Functions for serverless computation
  • Google Cloud Storage for featured game JSON hosting
  • Tailwind CSS for modern UI styling

This project was designed to show an end-to-end solution involving secure authentication, database operations, cloud integration, scalable architecture, and clean user interface design.
